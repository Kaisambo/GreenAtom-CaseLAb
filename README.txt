Доброго времени суток! В данной папке представлено решение задания ГринАтом. 

- Файл "StateTree.png" содержит визуализацию бинарного дерева статей, необходимого для решения поставленной задачи.

- Файл "GreenProject.exe" содержит консольное приложение, реализующее поставленную задачу на языке С++. (Для наглядной демонстрации работы программы рекомендуется запускать ее совместно с просмотром файла "StateTree.png").

- Файл "Теописание.docx" содердит теоретическую информацию о бинарном дереве, алгоритм реализации поставленной задачи, а также листинг кода на языке С++.

- Файл "Техническое задание.docx" содержит представление алгоритма в виде технического задания для передачи сторонней организации.

Благодарю за уделенное время на просмотр и оценку данной работы!
